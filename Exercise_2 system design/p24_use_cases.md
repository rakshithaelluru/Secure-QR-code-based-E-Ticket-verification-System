# Use Case Definition

The core use cases of the Secure QR Code–Based E-Ticket Verification System are:

- Register User – Allows the admin to create user accounts.

- Login User – Allows users to log into the system.

- Generate Ticket – Allows the system to create a digital ticket.

- Create QR Code – Generates a unique QR code for each ticket.

- View Ticket – Allows the customer to view their e-ticket.

- Scan Ticket – Allows the verifier to scan the QR code.

- Validate Ticket – Checks whether the ticket is valid or not.

- Mark Ticket as Used – Updates ticket status after successful verification.

- Manage Users – Allows admin to add or remove users.

- View Reports – Allows admin to view ticket usage reports.
