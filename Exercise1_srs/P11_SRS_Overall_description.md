# SRS Overall Description
# Product Perspective:
This system is a web-based application used for checking tickets using QR codes. It replaces manual ticket checking and makes the process faster and safer.

# Product Functions:
The system can:

- Create tickets with QR codes

- Store ticket details

- Scan QR codes

- Check if a ticket is valid

- Mark tickets as used

- Allow admin to manage users

# User Characteristics:

- Admin should know basic computer usage

- Verifier should know how to use a mobile phone

- Customer only needs to show the QR code ticket
- No special technical skills are needed.

# Constraints:

- Internet is required

- Camera device is needed to scan QR code

- System depends on server and database

- Data must be kept secure
