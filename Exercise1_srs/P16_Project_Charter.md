## Project Objective:
The objective of this project is to develop a system that verifies e-tickets using QR codes in a fast and secure way. The system aims to reduce ticket fraud, avoid duplicate ticket usage, and improve the speed of entry at theatres, transport systems, and events.

## Project Scope:
The project includes QR code generation, ticket storage, QR scanning, and ticket validation using a centralized database.

## Stakeholders:
Admin, Ticket Verifier, Customer, Event Organizers, and Management.
